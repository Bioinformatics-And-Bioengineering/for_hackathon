# プロジェクト起動手順


## バックエンド起動（Flask）

1. Python をインストール
2. バックエンドのディレクトリに移動

   ```bash
   cd backend
   ```
3. パッケージをインストール

   ```bash
   pip install -r requirements.txt
   ```
4. Flask アプリを起動

   ```bash
   python app.py
   ```
## フロントエンド起動 ##

1. フロントエンドに移動
2. ``` npm install ```
3. ``` npm start ```
